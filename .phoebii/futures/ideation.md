# Futures — Ideation Log
# -------------------------------------------------------
# Managed by /scope and reviewed by /futures
# Ideas land here when they are out of scope for the current phase.
# Nothing gets deleted. Ideas get parked, reconsidered, or archived.
# -------------------------------------------------------
# ⚠️  Do not manually reorder entries.
#     Chronological order is intentional — it tells a story.
# -------------------------------------------------------

## How This Works

When /scope catches an idea that drifts outside the current project phase
it parks it here automatically with context. When you are ready to revisit
parked ideas, run /futures and we will go through them together.

Status options:
- **parked** — logged, not yet reviewed
- **under_consideration** — being actively evaluated for pull-back
- **pulled_in** — moved back into scope, now in WBS
- **archived** — reviewed and deliberately set aside
